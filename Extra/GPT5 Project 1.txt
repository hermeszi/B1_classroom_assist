Project A (Team): ClassPulse AI
Outcome you can deliver in 2 hours
- Instructor types short notes (or uploads an audio file) for a student.
- Click “Generate” to get:
  - Parent-friendly lesson summary (email-ready).
  - Instructor-facing continuation notes (what to do next).
  - Structured fields: skills practiced, difficulty rating, engagement flags.
  - Optional: suggestions of next projects from a preloaded list.
- “Save & Send”:
  - Appends a row in a central Google Sheet (and optionally mirrors to per-student sheets).
  - Sends a formatted email to the parent via Gmail (Apps Script).

Proposed flow (MVP)
1) Instructor selects student from dropdown and types raw notes (or uploads audio).
2) Backend calls LLM to transform notes into structured JSON:
   - parent_email_subject
   - parent_email_html
   - internal_notes
   - skills (array)
   - difficulty (e.g., “Easy/Medium/Hard”)
   - next_project_recommendations (from a provided list)
3) Preview appears for quick edits.
4) Click “Save & Send”:
   - POST to Apps Script Web App.
   - Apps Script appends to a central “Entries” sheet and uses Gmail to send the parent email template.

Why Google Sheets + Apps Script now
- You avoid OAuth and SMTP complexity.
- You can reuse the school’s workflow (Sheets + Drive + Gmail).
- You gain “instant database” and easy email sending with the least plumbing.

- AI: Any hosted LLM (OpenAI/Groq/Anthropic). If you include audio transcription, use Whisper API.
- Data: Google Apps Script Web App writes to a central Google Sheet named “ClassPulse_Entries”.

# src/server.py
from fastapi import FastAPI, UploadFile, Form
from fastapi.middleware.cors import CORSMiddleware
import requests, os, json

app = FastAPI()
app.add_middleware(CORSMiddleware, allow_origins=["*"], allow_methods=["*"], allow_headers=["*"])

LLM_API_KEY = os.getenv("LLM_API_KEY")
GAS_URL = os.getenv("GAS_URL")  # Google Apps Script Web App URL

@app.post("/generate")
def generate(payload: dict):
    # Call your LLM provider here; below is pseudo:
    system = "You are an assistant ... JSON schema: {...}"
    user = json.dumps(payload)
    # response = call_llm(system, user)
    response = {
        "parent_email_subject": "Aiden’s Progress: Debugging and Loops",
        "parent_email_html": "<p>Today Aiden refined loops...</p>",
        "internal_notes": "Start next class with quick test cases ...",
        "skills": ["Loops", "Debugging"],
        "difficulty": "Medium",
        "next_project_recommendations": ["Fruit Sorter (tests)", "Traffic Light Simulator"]
    }
    return response

@app.post("/save")
def save(entry: dict):
    # Forward to Apps Script
    r = requests.post(GAS_URL, json=entry, timeout=10)
    r.raise_for_status()
    return {"status": "ok"}

// Code.gs
const SHEET_NAME = "ClassPulse_Entries";

function doPost(e) {
  const data = JSON.parse(e.postData.contents);
  const ss = SpreadsheetApp.getActive().getSheetByName(SHEET_NAME);
  if (!ss) throw new Error("Sheet not found");

  ss.appendRow([
    data.date_time,
    data.student_name,
    data.instructor_name || "",
    data.raw_notes || "",
    (data.skills || []).join(", "),
    data.difficulty || "",
